## 폴더 구조

```
LCK_study/
├── week01/
│   ├── README.md              # 이번 주 문제 목록 + 진도
│   ├── 0155-min-stack/
│   │   ├── README.md          # 문제 설명/링크 + 접근 메모
│   │   ├── lucian.py          # lucian은 .py 또는 .js
│   │   ├── kevin.py
│   │   └── chloe.java
│   └── ...

└── ...
```

- 문제 폴더명: `{문제번호 4자리}-{slug}` (예: `0155-min-stack`)
- 풀이 파일명: `{이름}.{확장자}` — lucian은 `.py`/`.js`, kevin은 `.py`, chloe는 `.java`
- 멤버가 늘면 폴더 안에 `{새이름}.{확장자}` 파일만 추가하면 됩니다.
