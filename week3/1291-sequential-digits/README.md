# 1291. Sequential Digits (Medium)

- **링크**: https://leetcode.com/problems/sequential-digits/description/
- **주제**: Enumeration / Math

## 문제 요약

각 자리 수가 이전 자리 수보다 정확히 1 많은 "순차 자리수"를 가진 정수를 찾는 문제. 주어진 범위 [low, high] 내의 모든 순차 자리수를 정렬된 리스트로 반환한다.

### 예시

```
입력: low = 100, high = 300
출력: [123, 234]

입력: low = 1000, high = 13000
출력: [1234, 2345, 3456, 4567, 5678, 6789, 12345]
```

### 제약

- 10 <= low <= high <= 10^9

### 포인트

- 순차 자리수의 최대 길이는 9 (1-2-3-...-9) 또는 0-1-2-...-8 정도로 제한된다는 점을 활용하면 가능한 모든 순차 자리수는 매우 적다.
- 가능한 시작 숫자와 길이 조합을 모두 생성해 필터링하는 접근이 효율적이다.
