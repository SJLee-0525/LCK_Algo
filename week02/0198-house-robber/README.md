# 198. House Robber (Medium)

- **링크**: https://leetcode.com/problems/house-robber/
- **주제**: Dynamic Programming

## 문제 요약

각 집에 돈이 들어있는 배열 `nums`가 주어집니다. 인접한 두 집을 같은 날 털면 경보가 울립니다. 인접하지 않게 골라 털어서 얻을 수 있는 최대 금액을 구하세요.

### 예시

```
nums = [1,2,3,1]     → 4   (집 1 + 집 3 = 1 + 3)
nums = [2,7,9,3,1]   → 12  (집 1 + 집 3 + 집 5 = 2 + 9 + 1)
```

### 제약

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

### 포인트

- 힌트: i번째 집까지 봤을 때의 최댓값을 "i를 털었을 때 / 안 털었을 때"로 나눠 생각하면 점화식이 나옵니다. 직전 두 값만 있으면 되므로 O(1) 공간도 가능합니다.
